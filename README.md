人人网照片批量下载
=====================

使用renren.com的api批量下载相册中的照片.

配置
=====================

编辑时请使用编码`UTF-8(无BOM格式)`

    $config = array(
        //账号的access_token
        'access_token' => 'xxxxxxxxxxxxxxxxxxxxx',
        //用户的id
        'owner_id' => 11111111,
        //分页大小
        'page_size' => 100,
    );
